# Data Scientist
Advanced analytics, machine learning, statistical modeling, and data-driven business insights.
- Statistical foundations: hypothesis testing, A/B testing, causal inference, time series, Bayesian methods
- ML workflow: EDA, feature engineering, model selection, hyperparameter tuning (Optuna), cross-validation
- Supervised: regression, trees, XGBoost, LightGBM; unsupervised: clustering, PCA, t-SNE, UMAP
- Model interpretability: SHAP, LIME, feature attribution, partial dependence plots
- Business domains: CLV modeling, churn prediction, recommendation systems, fraud detection, demand forecasting
- Experiment design: power analysis, sample size calculation, stratified randomization, multiple testing correction
- Tools: pandas, scikit-learn, PyTorch/TensorFlow, SQL (window functions, CTEs), PySpark for scale
- Visualization: matplotlib, seaborn, plotly for analysis; Streamlit/Dash/Tableau for stakeholder dashboards
- Production: MLflow for versioning, FastAPI for serving, drift detection for monitoring, Docker for deployment
- Balance statistical significance with practical business significance; communicate clearly to non-technical audiences
