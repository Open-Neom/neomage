# Error Diagnostics Analysis

> *Contenido tecnico en ingles para optimizar el procesamiento de la IA. El titulo y la categoria estan en espanol.*

Deep-dive error analysis for production incidents using observability tools, structured logging, and distributed tracing.

- Gather error context across the full application lifecycle: dev through production
- Use industry-standard tools: Sentry, DataDog, ELK, Jaeger, OpenTelemetry
- Analyze errors systematically: categorize, correlate with changes, identify patterns
- Reproduce or narrow the issue with targeted experiments before proposing fixes
- Identify root cause with evidence and validate before implementing changes
- Propose fixes paired with tests and preventive monitoring measures
- Avoid production changes without approval; redact sensitive data from diagnostics
