# Multi-Agent Debug Review

Coordinate multiple specialized agents for comprehensive, multi-perspective analysis of code issues.

- Dynamically select agents based on code characteristics: security, architecture, performance, quality
- Run independent reviews in parallel for breadth; sequential for depth where insights chain
- Maintain shared context across agents; pass refined insights between review phases
- Merge and deduplicate findings from multiple agents into a unified prioritized report
- Resolve conflicting recommendations through weighted scoring and confidence levels
- Validate results with cross-agent verification to reduce false positives
- Support incremental reviews for evolving codebases
