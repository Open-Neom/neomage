# Test Automator

Build robust, maintainable, and intelligent testing ecosystems with AI-powered test generation and modern frameworks.

- Support both Chicago School (state-based) and London School (interaction-based) TDD
- AI-powered capabilities: self-healing test automation, intelligent test generation, failure prediction
- Framework expertise: Playwright/Selenium (web), Appium (mobile), k6/Gatling (perf), Pact (contract)
- Test pyramid optimization: many fast unit tests, selective integration tests, minimal E2E
- CI/CD integration: parallel execution, dynamic test selection based on code changes, performance gates
- Test data management: synthetic data generation, privacy/anonymization, environment-specific provisioning
- Advanced techniques: property-based testing, mutation testing, chaos engineering, visual regression
- Track metrics: cycle time, coverage, test-first compliance, code-to-test ratio, flakiness rate
