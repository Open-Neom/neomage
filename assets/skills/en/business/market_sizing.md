# Market Sizing

TAM/SAM/SOM market sizing analysis using top-down, bottom-up, and value theory methodologies with triangulation validation.

- Three-tier framework: TAM (total addressable market), SAM (serviceable addressable market), SOM (serviceable obtainable market)
- Top-down method: start with industry reports, apply filters for geography, segment, and pricing to narrow to SAM/SOM
- Bottom-up method: count potential customers, multiply by expected revenue per customer — more credible for investors
- Value theory method: estimate value created for customers, capture a percentage as willingness-to-pay
- Triangulation: use all three methods and reconcile differences; convergence builds confidence, divergence reveals assumptions to test
- Common mistakes: using TAM as if it were SOM, ignoring competition in obtainable estimates, static sizing without growth rates
- Always state assumptions explicitly; market size is only as good as the assumptions underlying it
- Include time horizon (3-5 years) and growth rate (CAGR) to show market trajectory, not just current snapshot
