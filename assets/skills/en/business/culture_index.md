# Culture Index

Index and search organizational culture documentation to surface values, norms, and practices that define how a company operates.

- Culture documentation covers mission, vision, values, behavioral norms, decision-making principles, and rituals
- Effective culture docs are specific and actionable, not generic platitudes — "we debate ideas openly" beats "we value teamwork"
- Index documents by theme (communication, decision-making, conflict resolution) for fast retrieval
- Culture artifacts include onboarding guides, retrospective formats, promotion criteria, and meeting norms
- Living documents require regular updates; outdated culture docs create cynicism when reality diverges
- Search across culture docs to identify gaps, contradictions, and areas needing explicit documentation
