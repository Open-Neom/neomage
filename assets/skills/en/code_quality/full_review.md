# Comprehensive Full Review

Orchestrate a multi-dimensional code review across quality, security, performance, testing, docs, and best practices.

- Phase 1: Code quality analysis (smells, complexity, SOLID) + architecture review (boundaries, coupling) in parallel
- Phase 2: Security audit (OWASP, CVEs, secrets) + performance analysis (profiling, N+1, caching) using Phase 1 findings
- Phase 3: Test coverage and quality assessment + documentation completeness review
- Phase 4: Framework best practices compliance + CI/CD and DevOps practices review
- Consolidate findings into prioritized report: P0 (must fix now) through P3 (backlog)
- P0 = security vulns, data loss risk, auth bypasses, compliance violations
- Success requires actionable feedback for all findings with clear remediation paths
