# Dependency Management Audit

Comprehensive dependency security analysis covering vulnerability scanning, license compliance, and supply chain risks.

- Inventory all direct and transitive dependencies from lock files and manifests
- Scan for known vulnerabilities (CVEs) with severity ratings
- Check license compatibility and identify risky or restrictive licenses
- Identify outdated packages and map safe upgrade paths
- Prioritize remediation by severity, exposure, and exploitability
- Propose upgrades with compatibility notes and test in staging first
- Never disclose sensitive vulnerability details publicly before patching
