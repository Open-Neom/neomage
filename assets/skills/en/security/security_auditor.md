# Security Auditor

Comprehensive security auditing: DevSecOps, vulnerability assessment, compliance, and incident response.

- Audit workflow: confirm scope and assets, review architecture and threat model, scan, triage by severity, validate fixes
- DevSecOps pipeline: integrate SAST, DAST, dependency scanning, and secrets detection into CI/CD
- Shift-left: catch vulnerabilities early with pre-commit hooks, PR security checks, and developer training
- OWASP focus: assess against Top 10 (broken access control, injection, cryptographic failures, insecure design)
- Authentication audit: verify OAuth2/OIDC implementation, MFA enforcement, session management, JWT validation
- Authorization: test RBAC/ABAC enforcement, check for privilege escalation, verify least-privilege principles
- Cloud security: review IAM policies, security groups, encryption config, secrets management, container security
- Compliance mapping: GDPR, HIPAA, SOC2, PCI-DSS, ISO 27001 -- automate control validation where possible
- Findings format: severity (CVSS), business impact, exploit likelihood, remediation steps, residual risk
- Incident response: maintain playbooks, forensics procedures, breach notification workflows, recovery plans
- Supply chain: generate SBOM, monitor dependency vulnerabilities, verify SLSA provenance
