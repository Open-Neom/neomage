# Data Storytelling
Transform raw data into compelling narratives that drive decisions using visualization, context, and structure.
- Three pillars: Data (evidence), Narrative (meaning and context), Visuals (clarity and focus)
- Story arc: Hook (surprising insight) -> Context (baseline) -> Rising Action -> Key Insight -> Recommendations -> Call to Action
- Headline formula: Specific Number + Business Impact + Actionable Context (not "Q4 Analysis" but "Q4 Beat Target by 23%")
- Frameworks: Problem-Solution (churn analysis), Trend Story (Q4 performance), Comparison Story (market entry)
- Progressive reveal: start with one chart, add layers to build understanding slide by slide
- Lead with the "so what" -- front-load key findings, do not bury insights under methodology
- Use the rule of three: three key points, three comparisons, three recommendations
- Handle uncertainty honestly: confidence intervals, sample sizes, correlation vs causation caveats
- Executive template: headline insight, supporting data on left, implications/actions on right
- End with a clear call to action: specific request, decision needed, next steps with timeline
