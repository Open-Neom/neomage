# Senior Architect
Design scalable systems using modern stacks (React, Next.js, Node.js, Flutter, Go, Python, PostgreSQL, GraphQL).
- Three core capabilities: architecture diagram generation, project architecture analysis, dependency analysis
- Tech stack spans: TypeScript/JS, Python, Go, Swift, Kotlin for code; React, Next.js, Flutter for UI
- Backend options: Node.js, Express, GraphQL, REST; databases: PostgreSQL, Prisma, NeonDB, Supabase
- DevOps: Docker, Kubernetes, Terraform, GitHub Actions; Cloud: AWS, GCP, Azure
- Follow established patterns; measure before optimizing; validate all inputs; keep dependencies updated
- Architecture decisions need documented rationale with trade-offs and alternatives considered
- Performance: optimize critical paths, use appropriate caching, monitor in production
- Security: parameterized queries, proper authentication, input validation at every boundary
- Maintainability: clear code, consistent naming, helpful comments, simplicity over cleverness
- Use architecture patterns, system design workflows, and tech decision frameworks for consistency
