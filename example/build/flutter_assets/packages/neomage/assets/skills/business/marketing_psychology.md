# Marketing Psychology

Behavioral science applied to marketing with ethical guardrails, scored using the PLFS system for responsible persuasion.

- PLFS score = (Leverage + Fit + Speed + Ethics) minus Implementation Cost; ethics is a first-class scoring dimension
- Journey-based model selection: awareness (social proof, authority), consideration (anchoring, framing), decision (scarcity, loss aversion)
- Core principles: reciprocity, commitment/consistency, social proof, authority, liking, scarcity (Cialdini's six)
- Anchoring effect: the first number shown sets expectations — use strategic price anchoring in tier comparisons
- Loss aversion: people feel losses 2x more than equivalent gains — frame features as "don't miss out" not just "gain this"
- Social proof types: testimonials, user counts, case studies, trust badges, real-time activity notifications
- Ethical guardrails: never fabricate urgency, never manufacture scarcity, never exploit vulnerable populations
- Maximum 5 psychological models per recommendation to avoid manipulation stacking
