# Dimensionamiento de Mercado

- TAM (Total Addressable Market): mercado total si captuaras el 100% — establece el techo
- SAM (Serviceable Addressable Market): porcion del TAM que puedes servir con tu producto actual
- SOM (Serviceable Obtainable Market): porcion realista que puedes capturar en 2-3 anos

## Metodologias de calculo
- **Top-down**: partir de datos macro y filtrar. Ej: "mercado SaaS en LATAM" → % Mexico → % sector → % target
- **Bottom-up**: partir de unidades y escalar. Ej: clientes potenciales x precio promedio x frecuencia
- **Value-based**: cuanto valor genera la solucion al usuario x cuantos usuarios existen
- Usar ambos (top-down y bottom-up) y verificar que converjan; si divergen mucho, revisar supuestos

## Fuentes de datos
- Reportes de industria: Statista, IBISWorld, Gartner, Euromonitor
- Datos gubernamentales: INEGI (Mexico), Census Bureau, Banco Mundial
- Datos publicos: SEC filings, reportes anuales de competidores
- Datos propios: encuestas, analisis de trafico, datos de ventas

## Presentacion efectiva
- Mostrar supuestos explicitamente (cada numero tiene una fuente o calculo)
- Incluir escenarios: conservador, base, optimista
- Graficar funnel TAM → SAM → SOM para visualizar el filtrado
- Comparar con mercados analogos para validar orden de magnitud

## Anti-patrones
- TAM inflado citando mercados adjacentes no relacionados
- SOM sin plan concreto de como capturar esos clientes
- No citar fuentes de datos (numeros inventados)
- Ignorar competencia existente en el calculo de SOM
