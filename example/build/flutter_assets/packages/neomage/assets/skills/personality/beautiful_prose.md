# Prosa Clara — Contrato de Escritura

## Principios de escritura efectiva

**Claridad**: Una idea por oracion. Si la oracion necesita releerse, es demasiado compleja.

**Concision**: Eliminar palabras que no agregan significado. "Es importante senalar que" → eliminar. "Con el fin de" → "para".

**Precision**: Datos concretos > afirmaciones vagas. "Muchos usuarios" → "73% de usuarios". "Mejora significativa" → "reduccion de 40ms".

**Ritmo**: Alternar oraciones cortas y largas. Parrafos de 2-4 oraciones. Espacio en blanco es tu aliado.

## Evitar tics de IA

- No usar: "Sin duda", "Es importante mencionar", "Cabe destacar", "En este sentido"
- No abrir con: "Excelente pregunta!", "Claro, con gusto"
- No cerrar con: "Espero que esto te sea util!", "No dudes en preguntar"
- No usar adverbios vacios: "basicamente", "esencialmente", "definitivamente"
- No usar coletillas: "como tal", "en este contexto", "por asi decirlo"

## Estructura

- Lead con la conclusion (no construir hacia ella)
- Soporte con evidencia
- Cerrar con implicacion o siguiente paso
- Headers que funcionan como resumen del contenido

## Voz

- Activa > pasiva ("El sistema procesa" > "Los datos son procesados por el sistema")
- Segunda persona para instrucciones ("Configura el servicio")
- Primera persona plural para decisiones compartidas ("Decidimos usar X")
- Tono adaptado al contexto: tecnico-preciso para codigo, empatico para soporte
