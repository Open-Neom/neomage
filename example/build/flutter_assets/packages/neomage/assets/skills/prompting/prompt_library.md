# Prompt Library

Curated collection of battle-tested prompt templates for coding, writing, analysis, and creative tasks.

- Role-based prompts: Expert Developer (clean code, SOLID, pragmatic), Code Reviewer (bugs/structure/security, rated Critical/Suggestion/Praise), Technical Writer (simple language, examples, clear headings), System Architect (scalability/reliability/cost trade-offs)
- Task prompts: Debug (identify problem -> root cause -> fix -> prevention), Refactor (readability first, DRY, single responsibility), Write Tests (happy path, edge cases, error conditions, boundary values), API Documentation (endpoint, params, examples, errors, auth)
- Analysis prompts: Code Complexity (cyclomatic, coupling, cohesion, dependencies, tech debt), Performance (Big O, memory, I/O bottlenecks, quick wins), Security Review (input validation, auth, data protection, injections, dependencies)
- Creative prompts: Brainstorm Features (name, value prop, complexity, dependencies; rank by impact/effort), Name Generator (descriptive, evocative, acronyms, metaphorical)
- Core techniques: Chain of Thought (step-by-step reasoning), Few-Shot (input-output examples), Persona Pattern (role + traits + style + values), Structured Output (JSON schema specification)
- Prompt improvement checklist: clear objective, context provided, format specified, examples given, constraints defined, success criteria set
