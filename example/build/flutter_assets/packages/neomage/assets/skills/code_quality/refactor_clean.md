# Refactor and Clean Code

Improve code quality, maintainability, and performance through systematic refactoring aligned with SOLID and clean code principles.

- Assess code smells, coupling, and risky hotspots before touching anything
- Propose a refactor plan with incremental, reviewable steps
- Apply changes in small slices that keep external behavior stable
- Never change external behavior without explicit approval
- Update tests alongside code changes and verify no regressions
- Focus on reducing duplication, complexity, and tangled dependencies
- Keep diffs reviewable and changes reversible at every step
