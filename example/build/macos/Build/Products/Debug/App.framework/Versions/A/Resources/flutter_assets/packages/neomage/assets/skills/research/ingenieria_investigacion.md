# Ingenieria de Investigacion

- Aplicar rigor de ingenieria a procesos de investigacion: reproducibilidad, sistematizacion, automatizacion
- Todo experimento necesita: hipotesis, metodologia, datos, resultados, conclusiones
- Documentar el proceso, no solo los resultados; los callejones sin salida tambien son informacion valiosa

## Workflow de investigacion sistematica
1. Definir pregunta de investigacion precisa y acotada
2. Revisar estado del arte: que se sabe, que se ha intentado, gaps identificados
3. Disenar experimento/estudio con variables controladas
4. Recolectar datos con instrumentacion reproducible
5. Analizar con metodos apropiados al tipo de datos
6. Documentar hallazgos con suficiente detalle para reproduccion
7. Peer review: alguien mas revisa metodologia y conclusiones

## Herramientas de investigacion
- Gestion de papers: Zotero, Mendeley, BibTeX
- Notebooks reproducibles: Jupyter, Observable, R Markdown
- Version control para datos y codigo de analisis
- Pre-registro de hipotesis para evitar p-hacking

## Investigacion aplicada a producto
- User research: entrevistas estructuradas, encuestas, analisis de comportamiento
- Competitive analysis: feature mapping, pricing comparison, gap analysis
- Market research: TAM/SAM/SOM, tendencias, segmentacion

## Anti-patrones
- Cherry-picking datos que apoyan la conclusion deseada
- Cambiar hipotesis despues de ver resultados (HARKing)
- No documentar metodologia (investigacion no reproducible)
