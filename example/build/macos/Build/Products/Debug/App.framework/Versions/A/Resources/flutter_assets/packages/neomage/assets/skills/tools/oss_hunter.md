# OSS Hunter

Find high-impact open source contribution opportunities in trending repositories with actionable strategies.

- Phase 1 (Discovery): find trending repos via GitHub API; filter by stars >1000, recent activity, relevant topics
- Phase 2 (Issue Extraction): search for labels like help-wanted, good-first-issue, bug, roadmap
- Phase 3 (Feasibility Analysis): assess reproducibility, user impact, maintainer merge speed, and complexity
- Phase 4 (Dossier): generate structured report with project info, issue link, root cause analysis, proposed fix, confidence score
- Check recent PR history to gauge if maintainers actively merge community contributions
- Focus on issues with clear reproduction steps and well-defined scope for highest merge probability
- Prioritize issues that affect many users for maximum contribution impact
- Analysis is limited by context window for very large repositories; focus on relevant files
