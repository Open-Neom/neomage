# Financial Projections

Build 3-5 year financial projection models with cohort-based revenue, cost structures, and scenario analysis for startups.

- Cohort-based revenue model: group customers by acquisition month, apply retention curves and expansion rates per cohort
- Four cost categories: COGS (hosting, support), Sales & Marketing (CAC-driven), R&D (team), G&A (overhead) — each scales differently
- Three scenario framework: P10 (pessimistic, 10th percentile), P50 (base case), P90 (optimistic) with explicit assumption differences
- Key metrics to derive: CAC payback period, LTV/CAC ratio, burn multiple (net burn / net new ARR), gross margin trajectory
- Monthly granularity for Year 1-2, quarterly for Year 3-5; investors scrutinize near-term realism more than distant projections
- Revenue assumptions must be bottoms-up: number of sales reps x quota attainment, or traffic x conversion x ARPU
- Cost scaling rules: some costs are step functions (hiring), some are linear (hosting), some are fixed (rent) — model each correctly
- Cash flow timing matters: revenue recognition differs from cash collection; model both to avoid runway surprises
