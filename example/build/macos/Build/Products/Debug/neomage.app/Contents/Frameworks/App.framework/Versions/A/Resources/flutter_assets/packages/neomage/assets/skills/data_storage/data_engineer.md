# Data Engineer
Build scalable data pipelines, modern data warehouses, and real-time streaming architectures.
- Modern data stack: Fivetran/Airbyte (ingestion) + dbt (transform) + Snowflake/BigQuery (warehouse) + BI tools
- Batch processing: Spark, dbt, Airflow for orchestration; streaming: Kafka, Flink, cloud-native services
- Lakehouse architectures: Delta Lake, Apache Iceberg, Apache Hudi for ACID on object storage
- Data modeling: star schema for analytics, data vault for enterprise warehousing, OBT for simple analytics
- Data quality: Great Expectations for validation, dbt tests for transformation checks, freshness monitoring
- Orchestration: Airflow DAGs with task groups, SLA monitoring, incremental execution, retry with backoff
- Cloud platforms: AWS (Glue, Redshift, EMR), Azure (Synapse, Data Factory), GCP (BigQuery, Dataflow)
- Data governance: lineage tracking (DataHub/Atlas), catalogs, GDPR/CCPA compliance, PII protection
- Cost optimization: partition data, compress with Parquet/columnar, use spot instances for batch, lifecycle policies
- Prioritize reliability over speed; implement monitoring and alerting from the start
