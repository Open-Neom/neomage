# Senior Fullstack
Build complete web applications with React, Next.js, Node.js, GraphQL, PostgreSQL, and modern DevOps.
- Full stack: TypeScript/JS/Python/Go for code; React/Next.js/Flutter for frontend; Node.js/Express for backend
- Databases: PostgreSQL with Prisma or NeonDB/Supabase; DevOps: Docker, K8s, Terraform, CI/CD
- Code quality: follow established patterns, write comprehensive tests, document decisions, review regularly
- Performance: measure before optimizing, cache appropriately, optimize critical paths, monitor production
- Security: validate all inputs, parameterized queries, proper auth, keep dependencies updated
- Maintainability: clear code, consistent naming, helpful comments, simplicity first
- Use project scaffolding for consistent structure across new projects
- Analyze code quality with automated tools; fix issues before they accumulate
- Apply architecture patterns from reference docs; follow development workflow standards
- Cloud platforms: AWS, GCP, Azure with appropriate service selection per use case
